# [Pi](https://pi.dev/)

Set the below env variables for Pi to to pick up this config!!

```bash
export PI_CWD_ONLY=true
export PI_CODING_AGENT_DIR="$HOME/.config/pi"
```

## Plugins

```bash
pi install npm:pi-caveman
pi install npm:@krzyzanowskim/pi-profiles
pi-profile default /login
pi install npm:@juicesharp/rpiv-ask-user-question
pi install npm:pi-markdown-preview
```
